# 그룹 권한 평가

기록 작성자, 기록 대상자, 그룹 멤버십, 공유 대상 그룹이 서로 다를 때 의료 노트가 올바른 사람에게만 노출되는지 평가합니다.

## 핵심 질문

- 기록을 작성한 사람이 기록 대상자와 다를 때 권한이 올바른가?
- 저장 시 선택하지 않은 그룹에는 기록이 노출되지 않는가?
- 기록 대상자가 포함되지 않은 그룹에서는 해당 기록이 보이지 않는가?
- 같은 기록을 여러 그룹에 공유할 때 각 그룹 경계가 유지되는가?

## 실패 예시

- 내가 엄마 기록을 작성했다는 이유로 내 모든 그룹에 엄마 기록이 보임
- 엄마가 없는 친구 그룹에서 엄마의 진료 기록이 보임
- 가족 그룹에만 공유한 기록이 엄마/할머니 그룹에도 자동 노출됨
